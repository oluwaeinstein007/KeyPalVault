
## Installation

1. Clone the repository to your local machine:

```bash
git clone https://github.com/oluwaeinstein007/KeyPalVault.git
```
## Installation dependecies

1. Install composer:

```bash
composer install
```
2. Generate key:

```bash
php artisan key:generate
```
3. Run migration:

```bash
php artisan migrate
```
4. Genrate seeder:

```bash
php artisan db:seed
```
4. Start the local serve:

```bash
php artisan serve
```
